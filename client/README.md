# Client (Astro + Tailwind)

Frontend untuk undangan digital platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Server akan berjalan di `http://localhost:4321`

## Build

```bash
npm run build
```

## Environment Variables

Copy `.env.example` ke `.env`:

```bash
cp .env.example .env
```

Isi dengan:
- `PUBLIC_API_URL` - URL backend API (default: http://localhost:8787)

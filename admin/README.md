# Admin Dashboard (React + Vite)

Admin CMS untuk manage events & guests.

## Setup

```bash
npm install
cp .env.example .env
# Edit .env dengan Supabase credentials
```

## Development

```bash
npm run dev
```

Dashboard akan berjalan di `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- Dashboard overview (stats)
- Event management
- Guest management
- Generate invitation links
- RSVP monitoring
